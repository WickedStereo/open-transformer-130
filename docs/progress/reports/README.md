# Reports

This directory stores dated evidence for technical progress. Reports should be immutable snapshots tied to a run, milestone, or review event.

Recommended report types:

- repository audits
- architecture studies
- verification summaries
- FPGA bring-up logs
- synthesis, timing, power, DRC, and LVS reports
- silicon bring-up and benchmark reports
